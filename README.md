Локально развёрнутая вопросно-ответная система на базе Retrieval-Augmented Generation для доступа пользователей к содержанию оферты Wildberries. Позволяет задать вопрос боту [@you_wb_helper_bot](https://t.me/you_wb_helper_bot) на естественном языке и получить ответ. На данный момент бот не развёрнут на сервисе постоянно, доступ осуществляется по запросу по запросу.

Особенности:

  -  Полностью локальное развёртывание. Все компоненты — эмбеддер, реранкер, LLM, векторная БД, бэкенд, бот — работают на одной рабочей станции с обычной пользовательской GPU 5070ti. Никаких облачных API в рабочем пайплайне.
  -  Микросервисная архитектура. 6 контейнеров в Docker Compose, каждый отвечает за одну задачу. Любой компонент можно подменить правкой одной строки.

Навигация
 - Папки
   - api/ - финальный API и бот (используется в проде)
   - experiments/ - там лежит rag_exp_new.py, основной фреймворк
   - data/ - PDF-оферта (корпус)
   - context/docs_questions_qwen3_14b_awq.jsonl - сгенерированный бенчмарк 222 вопросов
   - new_results/ - финальные результаты всех экспериментов
   - demo - папка с демонстраицей работы бота
   - archive - папка со старыми, ненужными файлами
- Файлы:
   - docker-compose.yaml — главный (остальные docker-compose.*.yaml — экспериментальные/черновые)
   - Dockerfile.api
   - requirements-api.txt
   - rag_exp_new.ipynb — основной ноутбук экспериментов
   - create_test_set.ipynb — генерация бенчмарка (показывает методику создания тестов)

Стек технологий
  -  Python 3.12
  -  LlamaIndex — построение индекса, чанкирование, шаблонизация
  -  Qdrant — векторная БД с гибридным поиском
  -  vLLM — инференс эмбеддера и LLM (с поддержкой FP8 + paged attention)
  -  TEI (Text Embeddings Inference) — лёгкий сервер реранкера
  -  FastAPI — асинхронный бэкенд
  -  aiogram 3 — Telegram-бот
  -  Docker Compose — оркестрация
  -  RAGAS — оценка качества генерации

На данный момент список используемых моделей такой:
 - основная LLM - Qwen3-4B-Instruct (FP8)
 - embed модель - BGE-M3
 - rerank модель - BGE-reranker-v2-m3

Вся работа и тестирование системы выполнялись на конфигурации: Intel Core i7-14700KF · 32 ГБ RAM · NVIDIA RTX 5070 Ti 16 ГБ.


Быстрый старт
1. Клонирование репозитория
    ```bash
    git clone https://github.com/gosha22008/RAG.git
    cd RAG
    ```
2. Создайте файл .env в корне проекта. Поместите в него GEMINI_API_KEY (нужен только для этапа оценки метрик через библиотеку RAGAS) и TELEGRAM_BOT_TOKEN
3. Запуск всей системы
    ```bash
    docker compose up -d
    ```
Полное время холодного старта: 15–40 минут (большая часть — скачивание весов LLM). Время сильно зависит от скорости интернета.
